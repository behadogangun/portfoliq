💡 What is the Efficient Frontier?
The Efficient Frontier (Markowitz, 1952) shows the set of portfolios that offer the highest expected return for a given level of risk. Portfolios below the frontier are suboptimal — you can get more return for the same risk, or less risk for the same return. The Max Sharpe portfolio maximizes risk-adjusted return, while the Min Risk portfolio minimizes volatility.
